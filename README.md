# RawXchange

RawXchange is a responsive B2B raw-material marketplace website that connects buyers and sellers through a simple and user-friendly interface. The platform allows users to explore raw materials, view product information, add products to a shopping cart, manage quantities, and submit enquiries or seller registration forms.

## How the Website Works

1. Users open the home page and explore the available raw materials.
2. Products can be added to the shopping cart.
3. Cart information is stored in the browser using Local Storage.
4. Users can increase or decrease product quantities.
5. The cart automatically calculates the total price.
6. Buyers can submit product enquiries.
7. Sellers can access the seller registration and login sections.
8. Form information is submitted using FormSubmit.
